# Gkash USSD Service

A comprehensive USSD service for managing investment accounts, built with Express.js and integrated with Tiara Connect for SMS and USSD capabilities.

## Features

✨ **Account Management**

- Create investment accounts with OTP verification
- Multi-fund support (Money Market, Fixed Income, Balanced, Stock Market)
- Secure PIN-based authentication

💰 **Investment Operations**

- Invest in multiple fund types via PayHero M-Pesa checkout
- Withdraw from accounts via PayHero mobile money payout
- Check real-time balance
- Track transaction history

📱 **Multi-Channel Access**

- USSD support via Tiara Connect
- SMS notifications for all transactions
- RESTful API for mobile app integration

🔒 **Security Features**

- OTP-based account creation
- PIN-protected transactions
- Environment-based configuration
- Secure API credential management

---

## Tech Stack

- **Runtime**: Node.js v14+
- **Framework**: Express.js
- **SMS/USSD Provider**: Tiara Connect
- **Payment Integration**: PayHero (M-Pesa checkout and withdrawals)
- **Authentication**: OTP + PIN
